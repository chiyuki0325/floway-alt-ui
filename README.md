# Floway Alt UI

Experimental React Router frontend for Floway.

## Setup

This repository consumes Floway protocol/provider types from the Floway git
submodule at `vendor/floway`.

```bash
pnpm setup
```

For an already initialized checkout, `pnpm install` is enough.

## Development

```bash
pnpm dev
```

The dev server is available at `http://localhost:5173`.

## Build

```bash
pnpm typecheck
pnpm build
```
